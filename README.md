# SBG_VoiceOverlay
#### _A BepInEx5 mod to show the username and the volume to see who needs to kicked or muted._

## Installation

- [Install BepInEx 5](https://github.com/BepInEx/BepInEx/releases) 
- Download and drop the SBG-VoiceOverlay.dll file in the \BepInEx\plugins folder of your Super Battle Golf Installation



## Building for source
*set the env variable ``SUPER_BATTLE_GOLF_PATH`` to your Super Battle Golf installation directory.*
```sh
dotnet build
```
